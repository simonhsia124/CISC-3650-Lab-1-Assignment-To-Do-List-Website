A random To-Do List Website

How to use this to do list

    In order to add a task:
  • Write the task in the textbox and then click the green button that saids "Click here to add the task!"
  • Made a mistake and need to delete one task or all tasks?
  • 1. Click the yellow button that saids "Click here to delete a task!" to delete a single task
  • 2. Click the red button that saids "Click here to delete all tasks!" to delete all tasks
  • Reminder: The Yellow Button only deletes the most recent task you made and the Red button deletes all the tasks you have made
    Once a task has been made:
  • You have three things to do once a task is made:
  • 1. There is a checkbox next to the task you have created and you can only click that checkbox to indicate whether you have completed your task or not
  • 2. There is a way for you to set a due date of your task
  • 3. You can set the importance level of your tasks with 3 colors
  • (Red = Very Important Task) (Yellow = Moderately important) (Green = Not very important but should still be considered)
